1. Classes are ___templates_____ for creating objects. 
2. The _visibility_ of a property, a method or a constant can be defined by prefixing the declaration with the keywords public, protected or private. 
3. When a class derives from another class is called __inheritance___ 
4. _____Namespaces__ allow the PHP interpreter to tell the difference between two classes, functions, or constants that share the same name. 
5. Several predefined variables in PHP are __superglobal______, which means they are available in all scopes throughout a script. 
6. _____functions___ can be called directly, from within a script, to perform a specific task. 
7. The __header()___ function sends a raw HTTP header to a client. 
8. ____MVC(model-view-controller)____ is an architectural/design pattern that separates an application into three main logical components Model, View, and Controller. 
9. ___Libraries_____ allow developers to use code that they, or other programmers, have already written to perform a task.
